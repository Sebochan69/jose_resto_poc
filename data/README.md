# RestoPilot 1-Month Mock CSV Data

Generated fake but realistic restaurant operations data for 2026-06-01 to 2026-06-30.

Files follow the same schema as the current RestoPilot n8n POC data:

- sales.csv: 210 rows
- menu_items.csv: 7 rows
- inventory.csv: 12 rows
- payroll.csv: 162 rows
- expenses.csv: 105 rows
- settings.csv

Data notes:
- Weekend sales are higher than weekdays.
- Month has mild sales growth and promo/event bumps.
- Some menu items remain underpriced for AI insights.
- Inventory includes critical and watch stock levels.
- Payroll adds extra staff on high-demand days.
- Expenses include fixed rent, variable utilities/packaging, weekly marketing, and occasional maintenance.
